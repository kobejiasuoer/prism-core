# Prism Methodology Overview

Prism uses a two-stage workflow.

1. Stage 1 narrows the universe using structured scanning and scoring.
2. Stage 2 reviews shortlisted candidates and produces a structured report.

This public document describes the workflow shape, not the private production weights or thresholds.

## Public workflow

- Universe selection
- Demo scanning
- Demo scoring
- Deterministic public review
- Markdown reporting
